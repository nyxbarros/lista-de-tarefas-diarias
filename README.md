# GERENCIADOR DE TAREFAS PYTHON

## Rodar
```
# entrar na pasta com comando cd
python3 -m venv .venv
source .venv/bin/activate
python3 main.py
```

## Estrutura das listas

### Periódicas
```
a cada [quantidade de tempo] [dia/semana/mes/ano] ([data de quando deve renovar o ciclo, usando AAAA-MM-DD]):
- [ ] [nome da tarefa]
```

### Não periódicas
```
[nome da lista]:
- [ ] <(prazo da atividade, usando AAAA-MM-DD)> [nome da tarefa]
```
